## Usage Rules

- Keep attribution in article text, image alt text, or this document when adding new local assets.
- Prefer venue, object, tool, reading-space, or food photos over copyrighted screenshots/posters.
- If a Commons file warns about non-free elements or de minimis content, avoid using it as a cropped hero or thumbnail.
- Write original summaries and link to official pages rather than copying source text.
